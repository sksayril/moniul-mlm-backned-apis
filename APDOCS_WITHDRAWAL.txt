--------------------------------------------------
1. REQUEST WITHDRAWAL
--------------------------------------------------
Endpoint: POST /api/mlm/withdrawal/request
Description: Request a withdrawal from income wallet

Request:
```
POST /api/mlm/withdrawal/request HTTP/1.1
Host: localhost:3100
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
Content-Type: application/json

{
  "amount": 1000,
  "paymentMethod": "upi",
  "paymentDetails": {
    "upiId": "user@upi"
  }
}
```

OR

```
POST /api/mlm/withdrawal/request HTTP/1.1
Host: localhost:3100
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
Content-Type: application/json

{
  "amount": 1000,
  "paymentMethod": "bank",
  "paymentDetails": {
    "bankDetails": {
      "accountNumber": "1234567890",
      "ifscCode": "ABCD0001234",
      "accountHolderName": "John Doe",
      "bankName": "Example Bank"
    }
  }
}
```

Success Response (200 OK):
```
{
  "status": "success",
  "message": "Withdrawal request submitted successfully",
  "data": {
    "withdrawal": {
      "_id": "60d5ec9af11ab83456789012",
      "amount": 1000,
      "requestDate": "2023-06-15T08:30:00.000Z",
      "status": "pending",
      "paymentMethod": "upi",
      "paymentDetails": {
        "upiId": "user@upi"
      }
    },
    "remainingBalance": 4000
  }
}
```

Error Response (400 Bad Request):
```
{
  "status": "error",
  "message": "Minimum withdrawal amount is â‚¹150"
}
```

OR

```
{
  "status": "error",
  "message": "Insufficient balance for withdrawal"
}
```
